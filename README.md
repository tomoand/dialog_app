# 日記アプリ

Ruby on Rails に関して独学した成果をアウトプットするためにこのアプリを作成しました。  
今後も随時、学習内容をこのアプリにアウトプットしていこうと思っています。  
私の開発や勉強に関する進捗は[ブログ](https://tomoand.hatenablog.com/)に掲載しています。


## 使用技術
* Ruby 3.0.3
* Ruby on Rails 7.0.2.2
